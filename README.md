# Dear 銳銳 — v11 正式網址 / PWA 版

這版是可以部署到 Vercel / Netlify 的正式網站包。

## 新增
- `index.html` 正式入口
- `manifest.json`
- `sw.js` 離線快取
- App icon
- 手機可「加到主畫面」
- 保留 v10 的照片、備份碼、記憶庫、小紙條、模型切換

## 最簡單部署到 Vercel
1. 到 https://vercel.com
2. 新增 Project
3. 上傳整個資料夾或連 GitHub
4. Framework 選 Other
5. Deploy
6. 得到網址後，用手機 Chrome 開啟
7. Chrome 右上角 → 加到主畫面

## 換新版時
先在舊版按「產生備份碼」保存。
新網址打開後，把備份碼貼回去「套用備份碼」。
